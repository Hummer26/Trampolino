## FAQ

### Q1
```txt
Operate in tutta la Svizzera?
```

### A1
```txt
Sì. Possiamo organizzare le nostre attrazioni in diverse località della Svizzera, in base alla stagione, alla disponibilità e alle caratteristiche dell’evento.
```

### Q2
```txt
I prezzi sono indicati sul sito?
```

### A2
```txt
No. Ogni evento può avere esigenze diverse, quindi preferiamo preparare informazioni e preventivi su richiesta.
```

### Q3
```txt
È possibile richiedere l’attrezzatura per una festa privata?
```

### A3
```txt
Sì. Le nostre attrazioni possono essere richieste per feste private, compleanni, eventi aziendali, manifestazioni pubbliche e altre occasioni.
```

### Q4
```txt
Fornite anche un operatore?
```

### A4
```txt
Sì. Su richiesta possiamo fornire anche una persona dedicata alla gestione dell’attrazione durante l’evento.
```

### Q5
```txt
Il bungee trampoline è adatto anche ai bambini?
```

### A5
```txt
Sì, il bungee trampoline è pensato per bambini, ragazzi e adulti, nel rispetto dei limiti di peso e delle condizioni di utilizzo dell’attrazione.
```

### Q6
```txt
Quali sono i limiti di peso per il bungee trampoline?
```

### A6
```txt
Indicativamente l’attrazione è adatta a utenti da circa 15 a 85 kg. Le condizioni finali vengono sempre valutate in base alla situazione e all’attrazione installata.
```

### Q7
```txt
I castelli gonfiabili per che età sono indicati?
```

### A7
```txt
In generale sono indicati per bambini dai 3 ai 12 anni, ma l’idoneità può variare in base al tipo di struttura e all’evento.
```

### Q8
```txt
Cosa succede in caso di maltempo?
```

### A8
```txt
In caso di condizioni meteo non adatte, la disponibilità e l’utilizzo delle attrazioni devono essere valutati insieme. Contattaci per concordare la soluzione migliore.
```

### Q9
```txt
Quanto tempo prima bisogna contattarvi?
```

### A9
```txt
Consigliamo di contattarci il prima possibile, soprattutto per eventi, feste private o periodi di alta richiesta.
```

### Q10
```txt
Come posso richiedere informazioni?
```

### A10
```txt
Puoi chiamarci, scriverci su WhatsApp, inviare un’email o compilare il modulo di contatto sul sito.
```
